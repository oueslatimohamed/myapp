## Setup

First You have to clone it
```
$ git clone https://github.com/oueslatimohamed/MyApp.git
```
Then, to run this project, Type the code below to install it first:<br>
You can use docker:
```
$docker compose up
```
Or you can use NPM:
```
$npm install
```
after installing ,you gave to run it with:
```
$npm start
```
